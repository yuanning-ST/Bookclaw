# Calculator Skill

使用 calculator 工具执行精确的数学计算。

## 支持的运算
- add: 加法
- subtract: 减法
- multiply: 乘法
- divide: 除法

## 使用示例
- "计算 123 * 456" -> 使用 multiply
- "100 减去 37" -> 使用 subtract
- "帮我算一下 500 除以 8" -> 使用 divide
